To execute activitySelectionLTS.py, perform the following steps:
1.  Paste file 'activitySelectionLTS.py' to the directory of your choice.

2.  Navigate to directory chosen in step 1 (where activitySelectionLTS.py was pasted).

3.  Type 'python3 ./activitySelectionLTS.py' in command line (without quotes).

Note: If the above does not work, try typing 'chmod +x activitySelectionLTS.py' (without quotes),
	  then repeating step 3.
