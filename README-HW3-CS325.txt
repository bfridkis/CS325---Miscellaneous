To execute makingChange.py, perform the following steps:
1.  Paste file 'makingChange.py' to the directory of your choice.

2.  Navigate to directory chosen in step 1 (where makingChange.py was pasted).

3.  Type 'python3 ./makingChange.py' in command line (without quotes).

Note: If the above does not work, try typing 'chmod +x makingChange.py' (without quotes),
	  then repeating step 3.
